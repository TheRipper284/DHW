Prototipo IOT

Consta de un weareable para canes el cual tiene conectado un sensor de temperatura, un sensor de pulso cardiaco del can.

SENSOR DE PULSO CARDIACO

El sensor de pulso cardiaco permite realizar la medición del ritmo cardiaco de una persona utilizando microcontroladores, con únicamente tres terminales su sistema de conexión es bastante amigable (VCC, GND y señal de pulso). Con un Voltaje de alimentación: 5V.

SENSOR DE TEMPERATURA

El LM35 es adecuado para diversas aplicaciones incluidas electrodomésticos, teléfonos móviles, juegos, discos duros, gestión de baterías, instrumentos médicos portátiles e impresoras. Tensión de alimentación: 4 V a 30 V.
